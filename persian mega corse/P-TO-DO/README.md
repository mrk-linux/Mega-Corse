برنامه to-do 


ما با سه روش کامند, برنامه و تحت وب

برنامه to-do  را خواهیم ساخت 

to-do.py فایل نهای 

P-TO-DO.ipynb مراحل ساخت و پیشرفت برنامه


در ادامه، **ترجمه فارسی** و **انگلیسی** جمله شما ارائه شده است.

---

برای اینکه نسخه وب کار کند، ابتدا باید دستور زیر را اجرا کنید:

```bash
streamlit run "اسم برنامه"
```

اگر خطا (ارور) داد، قبل از اجرای برنامه، به محل پوشه برنامه بروید و **بعد** آن را اجرا کنید.

---

فرض کنیم اسم برنامه شما `app.py` است.

**کامل‌ترین روش اجرا:**

```bash
cd "مسیر/پوشه/برنامه"   # رفتن به پوشه برنامه
streamlit run app.py     # اجرای برنامه
```

**ترجمه این مثال:**

> ابتدا با دستور `cd` به پوشه برنامه بروید، سپس `streamlit run app.py` را اجرا کنید.
