
# 📄 تولید PDF از فایل CSV

این پروژه یک **سند PDF** با صفحات فرمت‌بندی شده، هدر، خطوط شبکه و فوتر بر اساس اطلاعات یک فایل CSV تولید می‌کند.

---

## 🚀 قابلیت‌ ها

- خواندن موضوعات و تعداد صفحات از فایل CSV
- ایجاد یک صفحه جداگانه برای هر موضوع
- افزودن عنوان، خطوط شبکه افقی و فوتر در هر صفحه
- پشتیبانی از موضوعات چند صفحه‌ای
- قابلیت تنظیم فونت، رنگ و چیدمان

---

## 📁 ساختار پروژه

```
PDF_Templates/
├── main.py               # اسکریپت اصلی برای تولید PDF
├── topics.csv            # فایل ورودی داده‌ها
└── output.pdf            # فایل PDF تولید شده
```

---

## 🛠️ پیش‌ نیازها

پکیج‌های مورد نیاز پایتون را نصب کنید:

```bash
pip install fpdf pandas
```

---

## ▶️ نحوه استفاده

اسکریپت را از ترمینال اجرا کنید:

```bash
python main.py
```

پس از اجرا، فایل `output.pdf` در همان پوشه ساخته می‌شود.

---

## 🧠 نحوه عملکرد

1. خواندن داده‌ها از فایل `topics.csv`
2. برای هر موضوع:
   - ایجاد صفحه جدید
   - اضافه کردن عنوان موضوع
   - رسم خط زیر عنوان
   - رسم خطوط شبکه افقی
   - اضافه کردن فوتر با عنوان موضوع
   - ایجاد صفحات اضافی در صورت نیاز
3. ذخیره فایل PDF نهایی با نام `output.pdf`

---

##  نکات

- مطمئن شوید فایل CSV در همان پوشه اسکریپت قرار دارد (یا مسیر فایل را به‌روز کنید)
- اسکریپت از سایز صفحه `A4` و واحد `mm` استفاده می‌کند
- می‌توانید فونت، رنگ‌ها و حاشیه‌ها را با ویرایش اسکریپت تغییر دهید
