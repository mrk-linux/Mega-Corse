# to-do app

We will build a **To-Do App** in **3 ways**:
1. Command Line App
2. GUI App
3. Web App

- **`to-do.py`** → Final application  
- **`P-TO-DO.ipynb`** → Steps to build and progress

---

##  Web Version

For the web version to work, first run the following command:

```bash
streamlit run "program_name"
```

> ⚠️ If you get an error, navigate to the program's folder first, then run the command again.

To stop Streamlit, press `Ctrl + C` in the terminal.

---

##  `pip freeze > requirements.txt`

This command takes the **list of all installed Python packages** in your environment and saves it into a file named `requirements.txt`.

- `pip freeze` → Shows the list of packages with their versions  
- `>` → Redirects the output to a file instead of displaying it in the terminal  
- `requirements.txt` → The destination file name

> ✅ **Usage:** When transferring your project to another environment, you can use this file to install all required packages at once.

---

##  Deploy & Get a Live URL

To get a **real URL** from the program:  
1. Run the program  
2. Enter the program's web page  
3. Click the **Deploy** option in the top-right corner  
4. The program will be built and a real URL will be generated

> ⚠️ **Important:** You must have uploaded the program folder to **Git** first.
